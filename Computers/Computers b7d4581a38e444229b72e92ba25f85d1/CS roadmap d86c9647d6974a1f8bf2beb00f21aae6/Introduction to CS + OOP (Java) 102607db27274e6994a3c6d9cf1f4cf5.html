      • important to remember when we declare primitive values (like int/numbers), in memory the name of the variable is store in binary and after the number itself in binary. so first we initialize number (int num = 5), memory is putting the name num in memory, then give us the the size for the number (4byets) and put the number 5 in binary in this space

      • String is object so when we create String/Object in memory there is the name of the object and then a pointer in different location in memory to the string itself. so first we initialize string (String str = “shahar”), memory is putting the name str in memory and point to different location to the string itself.
      • Important to remember is when we use object we always need to create instance of this object with the “new” keyword. With the word new we actually call the constructor function of the class.

      • Upcasting & Downcasting
        class  Parent{  
   void PrintData() {  
      System.out.println("method of parent class");  
   }  
}  
  
class Child extends Parent {  
   void PrintData() {  
      System.out.println("method of child class");  
   }  
}  
class UpcastingExample{  
   public static void main(String args[]) {  
        
      Parent obj1 = new Child();  
      Parent obj2 = (Parent) new Child();   
      obj1.PrintData(); // method of child class
      obj2.PrintData(); // method of child class
   }  
}
